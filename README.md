# 📰 HeadlineHub

HeadlineHub is a Spring Boot-based backend service that fetches real-time news headlines using NewsAPI and WebClient.

---

## 🚀 Feature

### feat(news): add APIs to fetch headline details

Fetch detailed news information including:

- Title
- Description
- Content
- Author
- URL
- URL To Image

Data is retrieved using **Spring WebClient** from **NewsAPI**.

---

## 🛠 Tech Stack

- Java 17+
- Spring Boot
- Spring WebFlux (WebClient)
- NewsAPI
- Maven
